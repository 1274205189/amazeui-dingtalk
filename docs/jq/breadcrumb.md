# 面包屑 Breadcrumb

面包屑导航组件。

## 默认分隔符

`````html
<ol class="breadcrumb breadcrumb-chevron">
  <li class="breadcrumb-item"><a href="http://amazeui.org">首页</a></li>
  <li class="breadcrumb-item"><a href="http://amazeui.org/css">分类</a></li>
  <li class="breadcrumb-item active">内容</li>
</ol>
`````

## slash 分隔符

`````html
<ol class="breadcrumb breadcrumb-slash">
  <li class="breadcrumb-item"><a href="http://amazeui.org">首页</a></li>
  <li class="breadcrumb-item"><a href="http://amazeui.org/css">分类</a></li>
  <li class="breadcrumb-item active">内容</li>
</ol>
`````

## angle 分隔符

`````html
<ol class="breadcrumb breadcrumb-angle">
  <li class="breadcrumb-item"><a href="http://amazeui.org">首页</a></li>
  <li class="breadcrumb-item"><a href="http://amazeui.org/css">分类</a></li>
  <li class="breadcrumb-item active">内容</li>
</ol>
`````

## 结合图标使用

`````html
<ol class="breadcrumb breadcrumb-chevron">
  <li class="breadcrumb-item"><a href="http://amazeui.org"><span
    class="icon icon-home"></span></a></li>
  <li class="breadcrumb-item"><a href="http://amazeui.org/css">分类</a></li>
  <li class="breadcrumb-item active">内容</li>
</ol>
`````
