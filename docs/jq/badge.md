# 小徽章 Badge

用于角标等辅助提示信息的组件。

## 默认样式

`````html
<span class="badge">badge</span>
`````

## 颜色

`````html
<span class="badge badge-primary">primary</span>
<span class="badge badge-secondary">secondary</span>
<span class="badge badge-success">success</span>
<span class="badge badge-warning">warning</span>
<span class="badge badge-alert">alert</span>
<span class="badge badge-dark">alert</span>
`````

## 椭圆形

`````html
<span class="badge badge-rounded">0</span>
<span class="badge badge-primary badge-rounded">1</span>
<span class="badge badge-secondary badge-rounded">2</span>
<span class="badge badge-success badge-rounded">3</span>
<span class="badge badge-warning badge-rounded">4</span>
<span class="badge badge-alert badge-rounded">5</span>
<span class="badge badge-dark badge-rounded">6</span>
`````
