# SearchBar

搜索组件。

## 组件

### SearchBar

`<SearchBar>` 组件。

#### Props

##### `component`

> PropType: `node`

组件容器元素，默认为 `form`。

##### `placeholder`

> PropType: `sting`

输入框占位符，默认为 `Search`。

##### `cancelText`

> PropType: `sting`

取消按钮文字，默认为 `Cancel`。

##### `onChange`

> PropType: `func`

输入框输入内容发生变化时的事件处理函数。

##### `onClear`

> PropType: `func`

点击清空输入框时的事件处理函数。

##### `onReset`

> PropType: `func`

点击「取消」按钮时的事件处理函数。
